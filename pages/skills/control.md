## Control

With this skill you have more control about where you *MyPet* walk and what the *MyPet* attacks.\\
To use this you have to rightclick with the **Control item** anywhere you want the *MyPet* has to walk to.\\
The item that is used to control the *MyPet* can be set in the config (`<color Coral>`Skill.Control.Item`</color>`)
